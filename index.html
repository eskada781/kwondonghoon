3402 권동훈 진로영어 세특




저는 수능특강 영어독해연습 1-6을 읽고, 웹사이트가 사업에 끼치는 영향에 대해서 생각해보면서도 한편 그러한 웹사이트가 어떻게 구축되는지 궁금해 했습니다.
그래서 웹사이트의 구축에 대해 조사하였고, 지금 보고 계신 것처럼 겉보기에는 형편없지만 웹사이트를 직접 구축해서 이렇게 보고서를 제출하게 되었습니다.

웹사이트는 html, css, 그리고 java script라는 컴퓨터 언어로 만들어집니다.
여기서 html은 웹사이트의 기반이 되는 언어이며, css는 웹사이트를 꾸미기 위한 언어로 이렇게 언어에 색을 주거나, 색을 더하는 등의 역할을 합니다.
그리고 java script(이하 js)는 두 언어가 다루지 못하는 기능을 다룹니다.
이렇게 세 가지 언어로 웹사이트는 작동되고, 여기에 더해 외부의 사진들을 추가하여 클릭 단추 등을 만들면 우리가 일반적으로 아는 웹사이트가 만들어 지게 됩니다.

이렇게 만들어진 웹사이트에 대한 정보는 인터넷을 통해 전해집니다.
이때 웹사이트에 대한 정보를 저장하고 접속하려는 컴퓨터에 그 정보를 보내는 컴퓨터를 서버, 반대로 이를 받는 컴퓨터를 클라이언트라 합니다.
코드로 이루어진 웹사이트의 정보는 크롬, 인터넷 익스플로러와 같은 웹 브라우저를 통해 시각화됩니다.
보통 회사들은 직접 서버 컴퓨터를 가지고 있어서 이를 가지고 직접 웹사이트를 운영하지만, 저같은 개인은 웹사이트 서버 운영을 대신해주는 호스팅이라는 방식을 사용합니다.
선생님이 읽고 계신 이 보고서는 github라는 호스팅 업체를 이용해 웹사이트의 형태로 전달된 것입니다.

다시 돌아와서, 직접 이렇게 웹사이트를 간략하게나마 구축해보며 웹사이트의 구축이 생각보다 어렵지 않다는 생각을 하게 되었습니다.
또한 디자인이 없는 이런 웹사이트를 보며 지문 속 내용인 웹사이트의 디자인 중요성을 직접 실감했습니다.
그리고, 오랜기간 준비했음에도 사진작업을 잘 하지 못해 디자인적으로 부족하게 웹사이트를 만든것이 아쉽게 느껴졌습니다.
